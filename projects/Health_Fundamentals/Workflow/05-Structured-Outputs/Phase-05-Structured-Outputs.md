# Phase 5: Structured Outputs

## Goal
Turn organized research into draft-ready deliverables.

## Technology Used
- `Open WebUI`
- `Ollama`
- Markdown documents
- Folder-based project storage

## What Happens In This Phase
- Theme briefs become outlines
- outlines become sections
- sections become usable documents

## Typical Outputs
- Book outline
- Chapter notes
- Research brief
- Contradictions report
- Business-plan draft
- Strategy memo

## Checklist
- [ ] Pick one output type
- [ ] Create a section outline
- [ ] Link each section to source-backed notes
- [ ] Mark unsupported areas clearly
- [ ] Draft one complete document from the first research batch

## Done Looks Like
- Research is no longer just collected; it is transformed into something publishable or actionable
