# Health Fundamentals Workflow

## Start Here
This folder structure is the operating map for the project.

Use it like this:

1. Read `Phase-Map.md`
2. Open the current phase folder
3. Complete the checklist in that phase
4. Move to the next phase only when the current one is stable

## Project Shape
The system is made of five major motions:

1. Get information into the system
2. Summarize and evaluate it
3. Organize it into themes
4. Turn it into usable documents
5. Orchestrate the whole workflow in a `.NET` app

## Current Direction
Open WebUI is the research brain.
Ollama is the local model engine.
The future `.NET` application is the workflow manager and file organizer.
