# Phase 4: Review And Clustering

## Goal
Move from simple summaries to higher-value research understanding.

## Technology Used
- `Open WebUI`
- `Ollama`
- Critical review prompts
- Folder-based note organization

## What Happens In This Phase
- Summaries are checked for weak points
- contradictions are flagged
- ideas are grouped into themes
- consensus and disagreement are separated

## Reviewer Roles
- `Critical Reviewer`
- `Theme Organizer`
- `Gap Finder`

## Example Theme Buckets
- Protein
- Recovery
- Sleep
- Energy balance
- Metabolism
- Training
- Habit formation
- Business strategy

## Checklist
- [ ] Create one critical-review prompt
- [ ] Create one clustering prompt
- [ ] Run contradiction checks on the first source batch
- [ ] Group notes into theme folders
- [ ] Create a short brief for each theme
- [ ] Mark uncertain claims for follow-up

## Done Looks Like
- You can see what the sources agree on, where they differ, and what needs more evidence
