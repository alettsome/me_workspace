## Quick Summary

Automatic background processing system that monitors `01-Inbox/` folder for PDF files, extracts text, chunks content, saves to database and filesystem, and creates notifications with processing metrics.

**Key Value**: Zero-friction intake - drop file, system handles everything automatically.
