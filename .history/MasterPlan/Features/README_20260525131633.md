# Features - Capability View

This folder organizes the system by **capabilities** (what exists), complementing the **Phases** view (when it was built).

## Structure

Each feature has:
- **Problem Definition**: What pain point does this solve?
- **Solution Approach**: How does it solve it?
- **Current Status**: Implemented, In Progress, Planned

## Features

- [Chunking](./Chunking/) - Split documents into processable pieces
- [Intake](./Intake/) - Bring content into the system *(planned)*
- [Summarization](./Summarization/) - Generate summaries from chunks *(planned)*
